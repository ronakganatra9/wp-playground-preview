WP Playground Preview

Version 1.0.0
✓ Extension Active

This extension adds "Try in Playground" and "Add to Playground list" to WordPress plugin and theme pages.

Features:

  • One-click testing in WordPress Playground
  • Add multiple plugins (and one theme) to your Playground list, then open Playground with all at once
  • Floating icon: view items list, remove any item from list, clear list, choose WordPress/PHP version (advanced), then Open in Playground

Visit any WordPress plugin or theme page to see the button in action!
